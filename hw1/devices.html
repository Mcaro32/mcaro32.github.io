Input & Output Devices

Input and output devices allow people to interact with a computer. Input devices send information to the computer, while output devices present information back to the user.

Without these devices, it would be impossible to enter commands, view results, hear audio, or print documents. They make computers useful for everyday tasks.

Common Examples

  • Keyboard
  • Mouse
  • Monitor
  • Printer
  • Speakers
  • Microphone
